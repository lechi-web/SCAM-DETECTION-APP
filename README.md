# ScamBuster AI 🚨

An AI-powered scam detection application that combines machine learning and LLM analysis to detect suspicious messages.

Features:
- ML-based spam classification
- LLM-powered scam explanation
- Phishing detection
- Scam pattern analysis
- Safety recommendations

Tech Stack:
- Python
- Streamlit
- Scikit-learn
- Groq LLM API
- NLP
